# employee-tracker
A very basic Employee Tracking system for nodeJS.

The employee tracker offers the following options:

## Create
* new Employees
* new Roles
* new Departments

## Read
* View All Employees 
* View Employees by Department
* View Employees by Manager
* View All Roles
* View All Departments
* Department Budgets

## Update
* Employee Role
* Employee Manager

## Delete
* Employees
* Roles
* Departments
